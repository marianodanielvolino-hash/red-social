# Red Social Web Site - Fundación

Este repositorio contiene el sitio web oficial de la **Fundación Red Social**, diseñado con una estética premium y arquitectura pública integrada.

## 🚀 Características
- **Think Tank:** Repositorio de documentos técnicos interoperables.
- **Portal Institucional:** Acceso dinámico a activos de gobernanza.
- **Formularios:** Conectados a Formspree para gestión de leads.
- **SEO Ready:** Optimizado para compartir en redes sociales con banners generados.

## 🔑 Acceso al Portal (Modo Demo)
Para visualizar el portal en entornos estáticos como GitHub Pages, use las siguientes credenciales:
- **Usuario:** `invitado@redsocial.org`
- **Contraseña:** (Cualquiera)

## 🛠️ Tecnologías
- HTML5 / CSS (Tailwind) / JavaScript
- PocketBase (Backend opcional)
- Formspree (Gestión de correos)
- Phosphor Icons
